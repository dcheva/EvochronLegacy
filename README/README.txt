Текущая версия перевода актуальна для игры Evochron Legacy версии не ниже v 1.0878.

Прежде, чем устанавливать перевод, необходимо выйти из игры!
Установка перевода:
--------------------
Распаковать содержимое архива "Mods-RUSsification" в корневую папку с игрой.
(например, D:\Steam\steamapps\common\Evochron Legacy\)
На вопрос "заменить файлы?" 
Отвечаем - копировать с заменой, ставим галочку "сделать это для следующих хх конфликтов файлов".

Ждем окончания, закрываем все окна, перезагружаем компьютер, запускаем игру и наслаждаемся игрой  Evochron Legacy  на русском, и с модом на HUD. ;-)
